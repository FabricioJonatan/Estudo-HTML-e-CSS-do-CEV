Utilizando links

Observações :

Para a criação de links externos é utilizado a tag A

Dentro do paramêtro HREF se coloca a URL do link desejado

Há dois paramêtros também que se pode adicionar :

TARGET="_BLANK" : faz com que o site seja abeerto em uma página em branco do navegador

REL="EXTERNAL" : indica que a página deverá ser aberta em uma nova aba do navegador

Links Externos na prática :

Você pode acessar ao YouTube através do meu site!!

E também para para um video especifico dele!!

Links Internos na Prática :

Links de Downloads na Prática :

A história da internet

Pra fazer um link de download é necessário adicionar o parametro DOWMLOAD para a tag A e nesse parametro adicionar o nome que o arquivo terá ao ser baixado

Também é necessário o parametro TYPE onde irá colocar o tipo do arquivo que será baixado, que deverá ser visto no iana.org
