Mais Formatações

Código Fonte e Pré-Formatação :

Dá para destacar códigos utilizando a tag CODE

Também é possível pré-formatar um texto para deixar da forma como foi escrita no código fonte utilizando a tag PRE 
        
EX :

    numero = int(input('Digite um numero :'))
    if numero > 10:
        print('Uau, que numero valioso')
    else:
        print('Poxa, esperava mais')

Citações Simples :

Como diria meu pai : Rapadura é doce mas não é mole

Para destacar citações simples, é usado a tag Q

Citações Completas :

Um dia, um antigo filósofo disse :

Nenhum prazer é em si um mal, porém certas coisas capazes de engendrar prazeres trazem consigo maior número de males que de prazeres.

Para dar essa ênfase maior nessa citação foi utilizada a tag BLOCKQUOTE e há o parâmetro cite nessa tag para indicar de onde a citação foi retirada
